## Quick Start
- Clone this repo 
```bash
git clone https://github.com/yifer97/ghs-translator.git
```
- Dependencies

推荐新建`conda`虚拟环境后使用`pip`安装
```
pyqt5==5.15
PyQtWebEngine==5.15
googletrans==2.4
```
