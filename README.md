## Installation
### 1. Clone this repo
```shell
git clone https://github.com/yifer97/ghs-translator.git
```
### 2. Dependencies 

> [Miniconda](https://docs.conda.io/en/latest/miniconda.html) envs is recommended

```pip
pyqt5==5.15
PyQtWebEngine==5.15
googletrans==2.4
```

### 3. Run
```shell
python ghs-translator.py
```
